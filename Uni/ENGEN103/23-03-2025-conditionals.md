# if statements

```matlab
if codition
    statements
end
```

## true or false

```matlab
num = 5;
if num
    disp("true")
end
```

> [!NOTE]
> The above code essentially checks for num's existence

## if or

```matlab
letter == 'y' || 'Y'
```

> [!NOTE]
> The above code will be true, the left side is false, but the right side is true

# while

```matlab
while codition
    statements
end
```
